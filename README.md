## 🚀 Sobre o Projeto

O **Cadastro de Restaurantes** foi desenvolvido com o objetivo de praticar lógica de programação e estruturação de sistemas simples no back-end.

O projeto simula um sistema básico de cadastro, permitindo armazenar e exibir informações de restaurantes de forma organizada.

---

## ⚙️ Funcionalidades

O sistema permite:

- 📥 Cadastrar novos restaurantes  
- 📋 Listar restaurantes cadastrados  
- 🏷️ Organizar informações de forma estruturada  
- 🧠 Trabalhar com manipulação de dados
